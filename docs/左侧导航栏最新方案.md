# 左侧导航栏最新方案

更新时间：2026-06-22

## 结论

左侧导航采用深蓝企业后台风格：一级模块是主模块卡片按钮，二级分组是弱标题，页面项是紧凑列表。多个一级模块可以同时展开，只有手动点击同一模块才收起。

## 规则

1. 左侧栏宽度固定为 `12.6rem`，不加宽、不缩窄。
2. 一级模块：经营中心、数据中心、财务中心、基础设置。
3. 一级模块显示图标、模块名、箭头；不显示“展开/收起”文字。
4. 展开状态用 `▾`，收起状态用 `▸`。
5. 点击一级模块只更新 `st.session_state["nav_open_modules"]`。
6. 点击页面按钮只更新 `st.session_state["nav_choice"]`，不改变已展开模块。
7. 当前页面所在一级模块使用弱蓝卡片高亮。
8. 当前页面使用半透明蓝底加左侧蓝色竖条。

## 实现位置

- 渲染函数：`render_sidebar`
- 导航结构：`NAV_MODULE_SECTIONS`
- 页面标签：`NAV_LABELS`
- 展开状态：`st.session_state["nav_open_modules"]`
- 当前页面：`st.session_state["nav_choice"]`
