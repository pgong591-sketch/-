# 经营汇总表：收入成本费用归集公式取数逻辑

整理日期：2026-06-13

来源公式文件：`C:\Users\pok\Desktop\收入成本费用明细表归集公式.xlsx`

公式解析文档：`docs/收入成本费用明细表归集公式解析.md`

## 一、定位

这套公式作为后续“经营汇总表”的取数来源和归集方式。

经营汇总表不应只展示原 Excel 模板，而应按以下逻辑生成：

```text
收入成本费用明细表原始数据
-> 明细项目识别
-> 经营归集规则
-> 经营汇总表指标
-> 驾驶舱展示 / 原表下钻 / 明细追溯
```

其中，收入成本费用明细表左侧数据是取数基础，右侧归集表是经营汇总表的公式来源。

## 二、数据链路

### 2.1 原始数据层

来源可以有两种：

1. 已上传的收入成本费用明细表。
2. 后续由科目余额表、损益明细表自动生成的收入成本费用明细数据。

系统字段建议统一为：

| 字段 | 含义 |
| --- | --- |
| `company_code` | 公司 / 校区编码 |
| `period` | 期间，例如 `202602` |
| `source_item_name` | 左侧明细项目名称 |
| `source_section` | 所属费用段或业务段 |
| `current_amount` | 本月金额 |
| `ytd_amount` | 本年累计金额 |
| `source_row` | 原 Excel 行号，可选 |
| `account_code` | 科目编码，可选 |
| `account_name` | 科目名称，可选 |

### 2.2 归集规则层

把 Excel 里的单元格引用改成系统规则：

```text
归集项目 = 若干明细项目金额之和
```

特殊项目：

```text
其他 = 成本费用合计 - 已明确归集项目合计
```

### 2.3 经营汇总层

经营汇总表直接读取归集后的结果：

| 经营汇总项目 | 取数方式 |
| --- | --- |
| 收入合计 | 来自收入成本费用明细表的收入合计 |
| 成本费用合计 | 来自收入成本费用明细表的成本费用合计 |
| 净利润 | 收入合计 - 成本费用合计 |
| 折旧及摊销 | 折旧费 + 待摊费 |
| 实际成本 | 成本费用合计 - 折旧及摊销 |
| 净利润（不含折旧与摊销） | 净利润 + 折旧及摊销 |

## 三、经营归集项目与取数公式

### 3.1 学生福利及教具

```text
学生福利及教具 = 学生福利 + 教学用具
```

对应 Excel 公式：

```excel
本月数 = C24 + C25
本年累计数 = E24 + E25
```

系统规则：

```text
source_item_name in ["学生福利", "教学用具"]
```

### 3.2 房租水电

```text
房租水电 = 房租 + 水电费
```

对应 Excel 公式：

```excel
本月数 = C26 + C27 + C74 + C75
本年累计数 = E26 + E27 + E74 + E75
```

系统规则：

```text
source_item_name in ["房租", "水电费"]
```

注意：房租、水电费可能出现在多个费用段，系统应按项目名称 + 费用段或行项目来源共同判断，避免漏算。

### 3.3 人工

```text
人工 = 工资 + 奖金 + 社保 + 社保费 + 公积金 + 福利费 + 教师福利
```

对应 Excel 公式：

```excel
本月数 =
C28+C29+C30+C76+C77+C78+C79+C56+C57+C58+C59+C23+C31+C54+C73

本年累计数 =
E28+E29+E30+E76+E77+E78+E79+E56+E57+E58+E59+E23+E31+E54+E73
```

系统规则：

```text
source_item_name in [
  "工资",
  "奖金",
  "社保",
  "社保费",
  "公积金",
  "福利费",
  "教师福利"
]
```

备注口径：

```text
工资、奖金、社保、公积金、福利
```

### 3.4 税金

```text
税金 = 发票税 + 税金及附加
```

对应 Excel 公式：

```excel
本月数 = C43 + C52
本年累计数 = E43 + E52
```

系统规则：

```text
source_item_name in ["发票税", "税金及附加"]
```

### 3.5 销售费用

```text
销售费用 = 业务宣传费
```

对应 Excel 公式：

```excel
本月数 = C70
本年累计数 = E70
```

系统规则：

```text

```

### 3.6 办公

```text
办公 = 办公费 + 维修费 + 通信费
```

对应 Excel 公式：

```excel
本月数 = C33+C62+C82+C37+C67+C60+C80+C86
本年累计数 = E33+E62+E82+E37+E67+E60+E80+E86
```

系统规则：

```text
source_item_name in ["办公费", "维修费", "通信费"]
```

备注口径：

```text
办公、维修、通讯、快递
```

后续可补充：

```text
source_item_name in ["办公费", "维修费", "通信费", "通讯费", "快递费"]
```

### 3.7 交际费

```text
交际费 = 招待费
```

对应 Excel 公式：

```excel
本月数 = C81
本年累计数 = E81
```

系统规则：

```text
source_item_name in ["招待费"]
```

后续可按管理口径确认是否改名为：

```text
差旅交际费 = 差旅费 + 招待费
```

当前样表右侧只归集了招待费。

### 3.8 折旧及摊销

### 3.9 管理中心


### 3.10 财务费用

```text
财务费用 = 财务费用
```

对应 Excel 公式：

```excel

```

```text

```

### 3.11 其他

```text
其他 = 成本费用合计 - 已明确归集费用合计
```

对应 Excel 公式：

```excel
本月数 = I20 - SUM(I9:I18)
本年累计数 = L20 - SUM(L9:L18)
```

系统规则：

```text
其他 =
成本费用合计
- 学生福利及教具
- 房租水电
- 人工
- 税金
- 销售费用
- 办公
- 交际费
- 折旧及摊销
- 管理中心
- 财务费用
```

注意：其他是剩余法，不从固定明细项目直接取数。

## 四、经营汇总表核心指标

### 4.1 收入合计

```text
收入合计 = 收入成本费用明细表中的收入合计
```

Excel 来源：

```excel
本月数 = C21
本年累计数 = E21
```

### 4.2 成本费用合计

```text
成本费用合计 = 收入成本费用明细表中的成本费用合计
```

Excel 来源：

```excel
本月数 = C112
本年累计数 = E112
```

### 4.3 净利润

```text
净利润 = 收入合计 - 成本费用合计
```

Excel 来源：

```excel
本月数 = I21 - I20
本年累计数 = L21 - L20
```

### 4.4 实际成本

```text
实际成本 = 成本费用合计 - 折旧及摊销
```

Excel 来源：

```excel
本月数 = I20 - I23
本年累计数 = L20 - L23
```

### 4.5 净利润（不含折旧与摊销）

```text
净利润（不含折旧与摊销） = 净利润 + 折旧及摊销
```

等价于：

```text
净利润（不含折旧与摊销） = 收入合计 - 实际成本
```

Excel 来源：

```excel
本月数 = I21 - I24
本年累计数 = L21 - L24
```

## 五、经营汇总表占比逻辑

### 5.1 占费用比例

```text
占费用比例 = 归集项目金额 / 成本费用合计
```

本月：

```text
month_expense_ratio = current_amount / current_cost_expense_total
```

本年累计：

```text
ytd_expense_ratio = ytd_amount / ytd_cost_expense_total
```

### 5.2 占收入比例

```text
占收入比例 = 归集项目金额 / 收入合计
```

本月：

```text
month_income_ratio = current_amount / current_revenue_total
```

本年累计：

```text
ytd_income_ratio = ytd_amount / ytd_revenue_total
```

## 六、建议植入经营汇总表的数据结构

后续可以把公式配置成规则表，而不是写死在页面里。

### 6.1 经营汇总归集规则表

建议表名：

```text
operating_summary_rules
```

建议字段：

| 字段 | 说明 |
| --- | --- |
| `id` | 主键 |
| `summary_item` | 经营汇总项目 |
| `source_item_name` | 来源明细项目名称 |
| `source_section` | 来源费用段，可选 |
| `formula_type` | `明细加总` / `剩余法` / `指标公式` / `占比公式` |
| `amount_scope` | `本月` / `本年累计` / `通用` |
| `sign` | `+` 或 `-` |
| `sort_order` | 展示顺序 |
| `remark` | 备注 |
| `status` | 是否启用 |

### 6.2 经营汇总结果表

建议表名：

```text
operating_summary_result
```

建议字段：

| 字段 | 说明 |
| --- | --- |
| `company_code` | 公司 / 校区 |
| `period` | 期间 |
| `summary_item` | 归集项目 |
| `current_amount` | 本月数 |
| `current_expense_ratio` | 本月占费用比例 |
| `current_income_ratio` | 本月占收入比例 |
| `ytd_amount` | 本年累计数 |
| `ytd_expense_ratio` | 累计占费用比例 |
| `ytd_income_ratio` | 累计占收入比例 |
| `formula_type` | 公式类型 |
| `source_detail` | 来源明细，可存 JSON |
| `created_at` | 生成时间 |

## 七、页面展示逻辑

经营汇总表页面建议按驾驶舱结构使用这些公式。

### 7.1 顶部指标卡

取数：

```text
收入合计
成本费用合计
净利润
净利润（不含折旧与摊销）
```

### 7.2 费用结构区

取数：

```text
学生福利及教具
房租水电
人工
税金
销售费用
办公
交际费
折旧及摊销
管理中心
财务费用
其他
```

展示：

```text
本月数
占费用比例
占收入比例
本年累计数
累计占费用比例
累计占收入比例
```

### 7.3 下钻逻辑

点击任意归集项目时，应展示来源明细。

例：

```text
点击“人工”
-> 展示工资、奖金、社保、社保费、公积金、福利费、教师福利
-> 按公司、期间、费用段、科目继续下钻
```

点击“其他”时：

```text
点击“其他”
-> 展示成本费用合计中未被明确归集的明细项目
```

```text
请根据 docs\收入成本费用明细表归集公式解析.md 和 docs\经营汇总表_收入成本费用归集公式取数逻辑.md，把收入成本费用明细表的右侧归集公式植入经营汇总表的数据取数逻辑。

要求：
1. 不要把 Excel 单元格地址硬编码到页面里，应抽象成“归集项目 -> 来源明细项目”的规则。
2. 经营汇总表的费用结构来源于收入成本费用明细表归集公式。
3. 支持本月数、本年累计数、占费用比例、占收入比例。
4. 支持核心指标：收入合计、成本费用合计、净利润、折旧及摊销、实际成本、净利润（不含折旧与摊销）。
5. “其他”使用剩余法：成本费用合计 - 已归集项目合计。
6. 保留后续下钻能力：每个归集项目可以追溯到来源明细项目。
7. 暂时先做规则和取数逻辑，不重做页面视觉。
```
