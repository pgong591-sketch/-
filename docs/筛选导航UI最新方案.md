# 筛选导航 UI 最新方案

更新时间：2026-06-22

## 结论

筛选导航统一使用白底卡片、紧凑主筛选行和可展开更多筛选区。

## 规则

1. 主筛选行展示：统计周期、经营单元、查询、重置、更多筛选。
2. 经营口径控件在主筛选行隐藏，后台默认使用默认公司口径。
3. 经营单元保留字段标题，删除下方“全部经营单元”说明。
4. 查询、重置、更多筛选按钮向下对齐输入框。
5. 更多筛选展开区向左对齐上方输入框。
6. 默认不显示快捷月份/快捷区间按钮。

## 实现位置

- 渲染函数：`_render_workspace_filter_bar`
- 更多筛选：`_render_more_filter_panel`
- 经营单元：`_render_company_unit_picker`
