# 202603 管理中心/非学科整合规则

## 1. 适用范围

- 管理中心部门收入成本费用表：`data/incoming/202603_mgmt_dept_income_cost.xlsx`
- 非学科管理中心部门收入成本费用表：`data/incoming/202603_non_subject_mgmt_dept_income_cost.xlsx`
- 非学科课酬：`data/incoming/202603_non_subject_teaching_fee.xlsx`
- 模板参考：`data/templates/报表模板.xlsx`
  - 第 7 个工作表：管理公司收入成本费用表
  - 第 8 个工作表：非学科管理中心收入成本费用表

## 2. 关键业务口径

1. 管理中心与非学科管理中心分别先做费用归纳，再按模板口径合并。
2. 管理中心收入合计不体现投资收益，按：
   - 主营业务收入 + 其他业务收入 + 营业外收入
3. 非学科课酬不是独立报表口径，作为“其中：校区上课收入”来源。
4. 部门对齐“可对齐则对齐，不强行对齐”。

## 3. 部门对齐规则

### 3.1 管理中心归纳展示列

- `董事会 -> 总经办`（与截图口径一致）
- `总经办 -> 集团办`（集团办与总经办对齐）
- `财务法务部 -> 财务部`
- `行政综合部 -> 行政教务部`
- `信息部`：预留列，当前为 0

### 3.2 非学科管理中心归纳展示列

- `素质管理中心 -> 执行委员会`（执行委员会与董事会同口径）
- `素质中心教研部 -> 教研部`
- `行政综合部 -> 行政教务部`
- `运营部 -> 咨询部`（保留为未强行并口径列）
- `学术中心 -> 学术中心`
- `信息部`：预留列，当前为 0

### 3.3 合并（管理中心成本分析）口径列

- 统一主列：`董事会、品牌部、财务法务部、教研部、行政教务部、新项目拓展部、总经办、人力服务部、幼儿园管理中心部、董事助理办公室、信息部`
- 未强行对齐列：`未对齐_咨询部、未对齐_学术中心`

## 4. 费用归纳与模板合并逻辑

### 4.1 管理中心归纳（模板行 94-103）

- 学生福利及教具
- 房租水电
- 人工
- 税金
- 销售费用
- 办公
- 差旅交际费
- 折旧及摊销
- 管理中心
- 财务费用
- 其他（按：成本费用合计 - 前述归类项）

指标行：

- 成本费用合计
- 收入合计（不含投资收益）
- 净利润
- 折旧及摊销（指标）
- 实际成本
- 净利润（不含折旧与摊销）

### 4.2 非学科管理中心归纳（模板行 92-101）

- 归纳项同上
- 其他按剩余法计算
- 指标行同管理中心
- 额外增加：`其中：校区上课收入`（来自非学科课酬）

### 4.3 管理中心成本分析（模板口径合并）

按模板第 7 表 124-136 行口径：

- 叠加项：`房租水电、人工、销售费用、办公、差旅交际费、其他`
- 保留管理中心原值项：`学生福利及教具、税金、折旧及摊销、管理中心、财务费用`
- 成本费用合计为上述项目求和
