## 一、表格结构判断

该文件是一张“左侧明细表 -> 右侧归集分析表”的公式样表。

左侧原始数据区：

- 范围：大致为 `A:E`
- `A` 列：明细项目名称
- `B` 列：本月单体金额
- `C` 列：本月合计，当前样表为 `=SUM(B行)`
- `D` 列：本年累计单体金额
- `E` 列：本年累计合计，当前样表为 `=SUM(D行)`

右侧归集结果区：

- 范围：`H:O`
- `H` 列：归集项目
- `I` 列：本月数
- `J` 列：占费用比例
- `K` 列：占收入比例
- `L` 列：本年累计数
- `M` 列：累计占费用比例
- `N` 列：累计占收入比例
- `O` 列：备注

## 四、占比公式

右侧归集表中，每个费用项目都有两类占比。

### 4.1 占费用比例

本月：

```excel
=本月项目金额 / 本月成本费用合计
```

例如学生福利及教具：

```excel
=I9/$I$20
```

本年累计：

```excel
=累计项目金额 / 累计成本费用合计
```

例如：

```excel
=L9/$L$20
```

### 4.2 占收入比例

本月：

```excel
=本月项目金额 / 本月收入合计
```

例如：

```excel
=I9/$I$21
```

本年累计：

```excel
=累计项目金额 / 累计收入合计
```

例如：

```excel
=L9/$L$21
```

## 五、系统落地建议

后续进入系统时，不建议硬编码单元格地址作为唯一规则，应转换成“归集项目 -> 明细项目关键词/科目/行项目”的规则表。

建议规则字段：

- `report_name`：收入成本费用明细表
- `summary_item`：归集项目，如人工、房租水电、办公
- `source_item_name`：左侧明细项目，如工资、奖金、房租
- `source_section`：所属费用段，可选
- `period_type`：本月 / 本年累计
- `formula_type`：明细加总 / 剩余法 / 利润公式 / 占比公式
- `sort_order`：展示顺序
- `remark`：备注

需要特别注意：

1. 同名项目会在不同费用段重复出现，例如工资、奖金、办公费、折旧费。
2. 仅按项目名称归集可能会误合并，最好结合行号、科目编码或费用段。
3. “其他”是剩余法，不是固定明细项。
4. “净利润（不含折旧与摊销）”当前公式写法是 `收入合计 - 实际成本`，与 `净利润 + 折旧及摊销` 等价。
