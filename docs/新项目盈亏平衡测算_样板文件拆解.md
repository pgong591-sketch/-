## 七、Word 方案补充：一对一服务项目测算逻辑

`盈亏平衡测算.docx` 进一步明确了一对一服务项目的测算逻辑，可直接作为第一版计算模型。

### 7.1 项目背景

该模型用于测算项目在不同收入结构、不同新招/续费占比、不同客单价条件下的：

- 保本收入
- 保本学生数
- 教师工资
- 规划师提成
- 总成本
- 利润情况

测算重点：

- 固定成本完整归集。
- 教师岗工资采用“保底工资与绩效工资取高”规则。
- 规划师提成按新招收入和续费收入分别计算。
- 最终输出月度盈亏平衡收入及对应学生数。

场地及运营固定成本：

| 成本项目 | 月金额 |
| --- | ---: |
| 租金 | 13,000 |
| 水电费 | 2,200 |
| 物业管理费 | 4,249 |
| 人工福利 | 8,800 |
| 运营费用 | 5,000 |
| 折旧摊销 | 9,000 |

人员配置：

| 岗位 | 人数 | 薪酬规则 |
| --- | ---: | --- |
| 运营负责人 | 1 | 固定工资 25,000 元/月 |
| 规划师 | 1 | 保底工资 2,380 元/月 + 招生/续费提成 |
| 教务专员 | 1 | 保底工资 2,380 元/月 |
| 教师岗 | 6 | 保底工资与绩效工资取高 |

基础固定成本：

- 场地及运营固定成本：42,249 元/月
- 运营负责人工资：25,000 元/月
- 规划师保底工资：2,380 元/月
- 教务专员保底工资：2,380 元/月
- 不含教师保底的固定成本合计：72,009 元/月
- 教师保底工资：6 × 9,000 = 54,000 元/月
- 最低刚性成本：126,009 元/月

### 7.3 核心计算公式

```text
基础固定成本 =
租金 + 水电费 + 物业管理费 + 人工福利 + 运营费用 + 折旧摊销
+ 运营负责人工资 + 规划师保底工资 + 教务专员保底工资

教师保底工资 = 教师人数 × 单个教师保底工资
教师绩效工资 = 月收入 × 教师绩效比例
教师实际工资 = max(教师保底工资, 教师绩效工资)

新招收入 = 月收入 × 新招收入占比
续费收入 = 月收入 × 续费收入占比
规划师提成 = 新招收入 × 新招提成率 + 续费收入 × 续费提成率

总成本 = 基础固定成本 + 教师实际工资 + 规划师提成
利润 = 月收入 - 总成本

平均提成率 = 新招收入占比 × 新招提成率 + 续费收入占比 × 续费提成率
边际贡献率 = 1 - 教师绩效比例 - 平均提成率

当盈亏平衡收入高于教师保底临界点时：
盈亏平衡收入 = 基础固定成本 ÷ 边际贡献率
```

### 7.4 默认场景

默认收入结构：

- 新招收入占比：50%
- 续费收入占比：50%
- 新招提成率：10%
- 续费提成率：5%
- 平均提成率：7.5%
- 教师绩效率：50%
- 总变动成本率：57.5%
- 边际贡献率：42.5%

默认盈亏平衡收入：

```text
盈亏平衡收入 = 72,009 ÷ 42.5% ≈ 169,433 元/月
```

默认输出结果：

| 项目 | 金额 |
| --- | ---: |
| 月收入 | 169,433 |
| 教师绩效工资 50% | 84,716 |
| 规划师提成 7.5% | 12,707 |
| 基础固定成本 | 72,009 |
| 总成本 | 169,433 |
| 利润 | 0 |

### 7.5 不同收入结构下的盈亏平衡点

| 新招收入占比 | 续费收入占比 | 平均提成率 | 边际贡献率 | 盈亏平衡收入 |
| ---: | ---: | ---: | ---: | ---: |
| 0% | 100% | 5.00% | 45.00% | 160,020 |
| 25% | 75% | 6.25% | 43.75% | 164,592 |
| 50% | 50% | 7.50% | 42.50% | 169,433 |
| 75% | 25% | 8.75% | 41.25% | 174,567 |
| 100% | 0% | 10.00% | 40.00% | 180,023 |

结论：新招收入占比越高，规划师提成率越高，盈亏平衡收入越高；续费收入占比越高，提成率越低，盈亏平衡收入越低。

### 7.6 保本学生数

保本学生数公式：

```text
保本学生数 = 盈亏平衡收入 ÷ 平均每生每月收费
```

按默认盈亏平衡收入 169,433 元/月测算：

| 平均每生每月收费 | 保本学生数 |
| ---: | ---: |
| 3,000 元/人/月 | 57 人 |
| 4,000 元/人/月 | 43 人 |
| 5,000 元/人/月 | 34 人 |
| 6,000 元/人/月 | 29 人 |
| 8,000 元/人/月 | 22 人 |

### 7.8 备案结论

在当前成本和薪酬规则下，若新招收入和续费收入各占 50%，则项目月度盈亏平衡收入约为：

- 169,433 元/月
- 即约 16.94 万元/月

若按平均每生每月收费 5,000 元测算，则保本学生数约为：

- 34 人

考虑退费、优惠、空课、教师产能不均、转化率波动等经营风险，建议项目实际经营目标收入不低于：

- 20 万元/月

该测算模型后期可作为项目管理系统、财务测算系统或经营看板中的盈亏平衡测算模块基础逻辑。
